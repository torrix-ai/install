# Torrix Java SDK

LLM observability for Java 11+ applications. Zero external dependencies.

## Installation

**Maven:**
```xml
<dependency>
  <groupId>ai.torrix</groupId>
  <artifactId>torrix</artifactId>
  <version>0.2.0</version>
</dependency>
```

**Gradle:**
```gradle
implementation 'ai.torrix:torrix:0.2.0'
```

## Quick Start

```java
import ai.torrix.*;
import com.openai.client.OpenAIClient;
import com.openai.client.okhttp.OpenAIOkHttpClient;
import com.openai.models.*;

import java.util.List;

public class Main {
    public static void main(String[] args) {
        // Initialize once at application startup
        Torrix.init(
            System.getenv("TORRIX_API_KEY"),
            "http://localhost:8088"
        );

        OpenAIClient client = OpenAIOkHttpClient.fromEnv();
        String userMsg = "What is the capital of France?";

        // Time your LLM call manually
        long start = System.currentTimeMillis();

        ChatCompletion response = client.chat().completions().create(
            ChatCompletionCreateParams.builder()
                .model(ChatModel.GPT_4O_MINI)
                .messages(List.of(ChatCompletionUserMessageParam.builder()
                    .content(userMsg)
                    .build()))
                .build()
        );

        long latencyMs = System.currentTimeMillis() - start;

        String reply = response.choices().get(0).message().content().orElse("");

        // Ingest the run -- returns immediately, never throws
        Torrix.ingest(IngestPayload.builder()
            .model(response.model())
            .provider("openai")
            .latencyMs(latencyMs)
            .inputTokens(response.usage().map(u -> (int) u.promptTokens()).orElse(0))
            .outputTokens(response.usage().map(u -> (int) u.completionTokens()).orElse(0))
            .prompt(userMsg)
            .response(reply)
            .status(200)
            .build());

        System.out.println(reply);
    }
}
```

## init()

```java
Torrix.init(String apiKey)
Torrix.init(String apiKey, String baseUrl)
```

Call once at application startup before any `ingest()` calls. Thread-safe. Calling `init()` a second time replaces the client.

| Parameter | Description |
|---|---|
| `apiKey` | Your Torrix API key (starts with `trxk_`) |
| `baseUrl` | Base URL of your Torrix instance. Defaults to `http://localhost:8088` |

## ingest()

```java
Torrix.ingest(IngestPayload payload)
```

Sends a run record to Torrix asynchronously on a background daemon thread. Returns immediately. Never throws. If `init()` was never called, the call is silently dropped.

## IngestPayload

Build with `IngestPayload.builder()...build()`. All fields are optional.

| Field | Type | Description |
|---|---|---|
| `runId(String)` | String | Unique run ID. Auto-generated by server if omitted. |
| `name(String)` | String | Human-readable label for this run. |
| `projectId(String)` | String | Project or namespace to group runs. |
| `provider(String)` | String | Provider name: `openai`, `azure`, `anthropic`, `sap`, `google`, etc. |
| `model(String)` | String | Model name, e.g. `gpt-4o-mini`, `claude-3-5-sonnet-20241022`. |
| `inputTokens(int)` | int | Number of prompt/input tokens. |
| `outputTokens(int)` | int | Number of completion/output tokens. |
| `thinkingTokens(int)` | int | Reasoning token count (extended thinking models). |
| `costUsd(double)` | double | Cost in USD. Torrix auto-calculates if omitted and model is known. |
| `latencyMs(long)` | long | End-to-end latency in milliseconds. |
| `status(int)` | int | HTTP status code of the upstream response, e.g. `200`, `429`, `500`. |
| `finishReason(String)` | String | Finish reason: `stop`, `length`, `tool_calls`, `content_filter`. |
| `prompt(String)` | String | Prompt text. Truncated to 800 characters. |
| `response(String)` | String | Response text. Truncated to 800 characters. |
| `error(String)` | String | Short error message if the LLM call failed. |
| `errorDetail(String)` | String | Full error detail or stack trace excerpt. |
| `thinking(String)` | String | Extended thinking / reasoning content. |
| `traceId(String)` | String | Groups multiple runs into an agent trace. |
| `sessionId(String)` | String | Groups runs into a multi-turn conversation session. |
| `agentName(String)` | String | Agent or step name within a trace. |
| `tags(List<String>)` | List&lt;String&gt; | Arbitrary labels for filtering runs in the dashboard. |

## How It Works

- `TorrixClient` maintains a `CachedThreadPool` of daemon threads named `torrix-ingest`.
- Daemon threads: the JVM will exit normally without waiting for in-flight ingest requests.
- Connect timeout: 5 seconds. Request send timeout: 5 seconds. All errors silently swallowed.
- JSON is serialized manually -- no Jackson, Gson, or any third-party library required.
- The `source` field is automatically set to `"sdk"` so your dashboard can distinguish proxy traffic from SDK traffic.

## Troubleshooting

**Runs not appearing in the dashboard**

1. Confirm `Torrix.init()` was called before `Torrix.ingest()`.
2. Confirm the `baseUrl` points to your running Torrix instance and is reachable from the JVM process.
3. Confirm the API key starts with `trxk_` and belongs to the correct project.

**JVM exits before ingest fires**

Daemon threads do not prevent JVM shutdown. For short-lived programs (e.g. scripts, tests), add a brief `Thread.sleep(500)` after the last `ingest()` call to let the background thread complete.

**`IllegalArgumentException` from `URI.create()`**

This means `baseUrl` contains invalid characters or a malformed URL. Confirm there are no trailing spaces and the URL includes the protocol (`http://` or `https://`).

## Spring AI Example

```java
@Service
public class ChatService {

    @PostConstruct
    void initTorrix() {
        Torrix.init(System.getenv("TORRIX_API_KEY"), "http://localhost:8088");
    }

    public String chat(String userMessage) {
        long start = System.currentTimeMillis();

        ChatResponse response = chatModel.call(new Prompt(userMessage));

        long latencyMs = System.currentTimeMillis() - start;
        String reply = response.getResult().getOutput().getContent();

        Torrix.ingest(IngestPayload.builder()
            .model("gpt-4o-mini")
            .provider("openai")
            .latencyMs(latencyMs)
            .prompt(userMessage)
            .response(reply)
            .status(200)
            .build());

        return reply;
    }
}
```

## LangChain4j Example

```java
ChatLanguageModel model = OpenAiChatModel.builder()
    .apiKey(System.getenv("OPENAI_API_KEY"))
    .modelName("gpt-4o-mini")
    .build();

Torrix.init(System.getenv("TORRIX_API_KEY"), "http://localhost:8088");

String userMessage = "What is the capital of France?";
long start = System.currentTimeMillis();
String response = model.generate(userMessage);
long latencyMs = System.currentTimeMillis() - start;

Torrix.ingest(IngestPayload.builder()
    .model("gpt-4o-mini")
    .provider("openai")
    .latencyMs(latencyMs)
    .prompt(userMessage)
    .response(response)
    .build());
```
